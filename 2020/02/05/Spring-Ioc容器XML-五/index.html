xml 中的同类型实例可以封装到list中

Xml自动装配

  • autowire属性仅限于自定义类型

    1. autowire=”default”/“no”: 不会自动赋值(默认)
    2. autowire=”byName”: 按照属性名赋值
      • private Car car; :根据 属性名作为id去容器中找到组件()赋值，找不到装配null

        car = ioc.getBean(“car”);
    3. autowire=”constructor”:
      • 按照有参构造器的参数进行赋值，没有直接为组件赋值(装配)null
      • 如果按照类型有多个，就按照属性名作为id继续匹配，找不到就null
